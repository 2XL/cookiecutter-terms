# cookiecutter-terms
cookiecutter terms and conditions

### Feature 
    
    Generate Markdown 
    
    Generate React component
    
    Generate PDF
    
    Generate Translations

### Dependencies


- react-pdf: for pdf generation


### Usage

clone and customize

```bash

git clone https://github.com/2XL/cookiecutter-terms

cd cookiecutter-terms
# create virtualenv
pip-sync

cookiecutter . --no-input -f

```
